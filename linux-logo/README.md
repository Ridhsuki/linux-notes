## Available Formats

The collection contains the following image formats:

* **SVG** — scalable vector images suitable for logos and high-resolution use
* **PNG** — raster images with broad application support
* **JPG** — compressed raster images suitable for previews and general use

## Usage

Clone or download this repository, then use the required image through its relative path.

Example in Markdown:

```markdown
![Tux Linux Mascot](./Tux.svg)
```

Example in HTML:

```html
<img src="./Tux.svg" alt="Tux Linux Mascot" width="200">
```
